# Raw+TF encoder downstream generation comparison

This S24 development diagnostic compares VI-only, full VS→VI, and gated
residual raw+TF encoders with one frozen SD1.5 UNet+LoRA generator. Because the
raw+TF latent is 256-dimensional while the historical EEG projector expects
512 dimensions, each encoder receives a separately trained bridge with the
same initialization, data split, optimizer, seed, and validation rule. The
UNet and LoRA parameters never update.

The bridge is selected by deterministic VI-validation diffusion loss. The
selected bridge is evaluated once on the same two test trials per class using
identical diffusion noise. Correct, next-class shuffled, and zero tokens are
generated to measure accuracy, collapse, and conditioning sensitivity.

This is a downstream utility diagnostic, not an independent confirmatory test:
S24 is the development subject and only 18 test EEG trials are generated by
default.
